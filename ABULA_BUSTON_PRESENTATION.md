## How It Works: Your Daily Routine

```
1. Staff Login in Morning
   → Use simple access code

2. Cashier Takes Orders
   → Customer orders food/drinks
   → Click items on tablet/computer
   → Customer pays
   → Print receipt

3. System Records Everything
   → Sales tracked automatically
   → End of shift summary ready

4. You Check Dashboard
   → See today's sales anytime
   → Get daily report
   → Plan for tomorrow
```
